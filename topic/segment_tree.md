# Segment Tree (线段树)

给定一个长度为 $n$ 的数组 $A$，支持以下操作：

- `query(l, r)`：查询区间 $[l, r]$ 的和，其中 $0 \leq l \leq r \leq n-1$
- `update(i, v)`：将 $A[i]$ 的值更新为 $v$，其中 $0 \leq i \leq n-1$

如果使用一维数组实现


树状数组主要用于单点更新和区间查询，而线段树可以支持区间更新和区间查询。
